# Recruitment Agency Website

Static recruitment agency website ready for GitHub Pages.

Pages:
- Home
- About
- Services
- Contact

Includes a contact form with routing options (Clients, Candidates, Student Service, General Enquiries).
Uses EmailJS for direct email sending from a static site.

## Publish to GitHub Pages
1. Create a new GitHub repository
2. Upload all files from this folder
3. Go to Settings → Pages
4. Select main branch and root folder
5. Save — site will be live

## Contact Form Setup
Sign up at https://www.emailjs.com  
Replace placeholders in `js/main.js` with your EmailJS keys.
